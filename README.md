## 🏗️ 三层架构

本项目包含完整的三层架构演示：

- **第一层**: MinIO - 对象存储（图片、视频、文档）
- **第二层**: PostgreSQL - 元数据（消息记录、用户信息）
- **第三层**: Milvus - 向量数据库（语义搜索）

详见：[三层架构文档](./docs/THREE-TIER-ARCH.md)

## 🚀 快速开始

### 1. 启动三层架构

```bash
cd ~/ai-agent-learning/three-tier-arch
./start.sh
```

### 2. 启动 Langchain-Chatchat

```bash
cd ~/ai-agent-learning/langchain-chatchat
pip install -r requirements.txt
python startup.py
```

### 3. 开始学习

按照每日计划逐步完成：[每日任务](./docs/DAILY-PLAN.md)
